# Sistema Fiscalía Flagrancia

Proyecto PHP simple y funcional con MVC básico.

## Requisitos
- XAMPP con Apache y MySQL activos
- PHP 8+
- Base de datos `flagrancia_fiscalia`

## Instalación
1. Copia la carpeta `Semana7_Castro_Miranda_Quispe` en `C:\Doger\Xampp\htdocs\`.
2. Importa `config/BD.sql` en phpMyAdmin.
3. Abre:

```txt
http://localhost/Semana7_Castro_Miranda_Quispe/public/
```

## Usuario inicial
- Usuario: `admin`
- Contraseña: prueba con la que ya te permitió entrar. Si necesitas cambiarla, usa `public/clave.php?clave=NUEVA_CLAVE` y actualiza el hash en la tabla `usuarios`.

## Funciones incluidas
- Login y logout
- Dashboard
- Registro de carpetas
- Listado con filtros
- Edición y eliminación
- Exportación CSV
- Gestión simple de usuarios admin
